# PulseFX Scanner website concept

This folder contains a polished single-page website concept for a forex-first market dashboard.

## Included
- `index.html` — responsive static landing page / concept mockup

## Concept summary
The page is structured as:
1. **Hero section** — headline, positioning, CTA, and value props
2. **Market snapshot preview** — current-price style cards for major FX pairs plus optional gold
3. **Top opportunities** — top 5 buy opportunities and top 5 sell opportunities with scores and setup rationale
4. **Why it works** — section explaining the product framing and scanner methodology
5. **Disclaimer** — clear non-advisory language
6. **FAQ / trust copy** — transparency and expectation-setting
7. **Footer CTA** — closes with a professional product-style message

## Positioning guidance
- Keep the language centered on **market information**, **setup scanning**, and **trade preparation**
- Avoid phrasing that implies guaranteed results, certainty, or personalized financial advice
- Forex is the primary focus; **XAU/USD (gold)** is included as an optional add-on market
